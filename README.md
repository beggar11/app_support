# App Store Connect 技术支持

一个全面的 App Store Connect 技术支持网站，帮助 iOS 开发者解决 App 提交、审核、证书配置、TestFlight、内购等常见问题。

## 功能

- **搜索** — 快速搜索所有指南、问题和 FAQ
- **操作指南** — App 提交、TestFlight、内购、证书配置等详细步骤
- **常见问题排查** — 按类别查找签名、审核、上传、内购、账户问题的解决方案
- **FAQ** — 20+ 常见问答
- **错误代码速查** — ITMS 错误码快速参考表
- **官方资源链接** — Apple 开发者文档、审核指南、API 文档等

## 本地预览

直接用浏览器打开 `index.html`，或使用任意静态服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

## 部署

支持部署到 GitHub Pages、Netlify、Vercel 等静态托管平台。纯静态站点，无需构建步骤。
